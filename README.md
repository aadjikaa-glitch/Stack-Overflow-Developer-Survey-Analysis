# Stack-Overflow-Developer-Survey-Analysis
This project focuses on analyzing the Stack Overflow Developer Survey dataset using Python and Pandas. The analysis includes data cleaning, exploratory data analysis (EDA), and insights into developers’ experience, technologies, work formats, salaries, and Python usage trends.

## Project Tasks

* Loading and processing CSV datasets
* Data cleaning and handling missing values
* Analyzing survey participant data
* Checking response completeness
* Calculating statistical metrics (`mean`, `median`, `mode`)
* Analyzing developers’ work formats (`Remote`, `Hybrid`, `Office`)
* Exploring Python popularity among respondents
* Calculating the percentage of developers using Python
* Analyzing learning methods used by developers
* Comparing Python developer salaries across countries
* Identifying respondents with the highest salaries
* Exploring Python usage across different age groups

## Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Jupyter Notebook

## Datasets

The project uses the annual Stack Overflow Developer Survey datasets:

* `survey_results_public.csv`
* `survey_results_schema.csv`
